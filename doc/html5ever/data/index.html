[−][src]Module html5ever::data

Data that is known at compile-time and hard-coded into the binary.

Statics

C1_REPLACEMENTS

The spec replaces most characters in the ISO-2022 C1 control code range (U+0080 through U+009F) with these characters, based on Windows 8-bit codepages.

NAMED_ENTITIES

A map of entity names to their codepoints. The second codepoint will be 0 if the entity contains a single codepoint. Entities have their preceeding '&' removed.
